## Get to develop MathCalc

1. Install dependencies

   ```bash
   npm install
   ```
2. Add your environment variables into env-var.ts (Change env-var-example.ts to env-var.ts)

3. Make your code

4. Start the app

   ```bash
    npx expo start
   ```
